\doxysubsubsection{Detailed Description}
The \mbox{\hyperlink{a00892}{Sensor}} class represents the air-\/quality sensors stationed in Augsburg. Each sensor collects data, has a failure rate, a location, and is part of one, none or multiple categories. The \mbox{\hyperlink{a00892}{Sensor}} class also provides access to these data points. In the E-\/\+Mail-\/\+Notification System, this class is mainly used organize and transport data collected by and about sensors. This information is then needed for report-\/ and alert-\/mails. The \mbox{\hyperlink{a00892}{Sensor}} class does not hold information about the subscribers to a sensor, as the subscriptions are handled in a distinct \mbox{\hyperlink{a00900}{Subscription}} class.
