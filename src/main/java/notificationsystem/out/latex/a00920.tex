\doxysubsubsection{Detailed Description}
The \mbox{\hyperlink{a00920}{Mail\+Builder}} class is responsible to build the different types of e-\/mails needed, alert, report and confirmation mail, from the relevant information provided to it by the method-\/parameters. The class provides one method for each of the e-\/mail type, each returning a finished mail of the relevant type.
