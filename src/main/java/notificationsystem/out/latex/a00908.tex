\doxysubsubsection{Detailed Description}
\mbox{\hyperlink{a00908}{Alert}} e-\/mails are used to notify subscribers of sensors of a malfunction of that sensor. The \mbox{\hyperlink{a00908}{Alert}} class contains the e-\/mail addresses of the sender and the receiver, as well as a subject and a message.
