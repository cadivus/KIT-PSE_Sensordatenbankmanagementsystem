\doxysubsubsection{Detailed Description}
The abstract \mbox{\hyperlink{a00916}{EMail}} class represents a normal e-\/mail. All other e-\/mail types inherit from it. It contains the basic parts of an e-\/mail\+: sender, receiver, subject, message and attachment. The class furthermore provides getters and setters for these attributes.
