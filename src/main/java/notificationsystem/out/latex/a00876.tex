\doxysubsubsection{Detailed Description}
An instance of the \mbox{\hyperlink{a00876}{Checker\+Util}} class always runs in the background. Its purpose is to regulary check if alerts or reports have to be sent by the system. The class is designed with the singleton-\/pattern as only a single instance should, for example, issue reports.
