\doxysubsubsection{Detailed Description}
The \mbox{\hyperlink{a00896}{Sensor\+DAO}} class implements the \mbox{\hyperlink{a00888}{DAO}} interface to handle database queries regarding sensors. For that end it provides get, save, delete and get\+All methods designed to hide the actual database queries, offering a single access point to all sensor related data and information.
