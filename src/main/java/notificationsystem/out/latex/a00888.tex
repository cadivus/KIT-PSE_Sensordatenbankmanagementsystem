\doxysubsubsection{Detailed Description}
The \mbox{\hyperlink{a00888}{DAO}} (Data-\/\+Access-\/\+Object) interface provides an interface with the basic get, save, delete and get\+All methods. The classes implementing it are used to hide away specific queries to the database ans subsequently simplify database accesses.
