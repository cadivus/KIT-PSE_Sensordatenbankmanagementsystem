\doxysubsubsection{Detailed Description}
\mbox{\hyperlink{a00924}{Report}} e-\/mails are sent to subscribers of a sensor at regular intervals dependant on the time of subscription and user adjustments. The reports contain the data collected by the sensor over the last time period as well as potentially interesting metadata such as failure rate over the last timer period. The \mbox{\hyperlink{a00924}{Report}} class represents such a report e-\/mail. It contains the e-\/mail addresses of sender and receiver, as well as a subject and a message. Lastly the e-\/mail includes an attachment with the data collected by the sensor.
