Detailed Description

The DAO (Data-Access-Object) interface provides an interface with the basic get, save, delete and getAll methods. The classes implementing it are used to hide away specific queries to the database ans subsequently simplify database accesses.

Parameters
<T>The interface as well as all methods use the basic T-parameter as different implementing classes are used to access different kinds of objects.
